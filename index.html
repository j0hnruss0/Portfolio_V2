Liri-Node Search

Made with Node JS

Click here to check it out!

An easy to use CLI search tool for finding your favorite songs on Spotify, looking for upcoming concerts for bands that play your favorite songs, or even learning more about movies that you have/want to see! Written in Node.js, powered with Node Spotify API, Axios, Dotenv, and Moment.

Commands start with "node liri", followed by:

  • "concert-this (band name)"
  • "movie-this (movie name)"
  • "spotify-this-song (song name)"

Word Guess 2

Made with Node JS

Click here to check it out!

Similar to my last JavaScript Word Guess game, but this one runs with Node.js in the CLI terminal. This is my interpretation of Hangman, the game starts with a word with each letter hidden by blanks. Enter a letter into the command line and hit "enter". If you're right, the blanks will update accordingly but if you're wrong 9 times, the game will end and you will lose. If you guess the show with less than 9 wrong guesses, YOU WIN! If you would like to try it out yourself, git clone the files onto your machine and "npm install" the node_modules into the same folder

The game starts with the command "node index"

Bamazon Store

CLI App, powered with Node and MySQL. Installed packages include mysql and inquirer

Click here to check it out!

Start up the app by entering "node bamazonCustomer" and you will see a list of items for sale. Enter the number to the left of the item you want in the CLI, then enter how many you want to buy. If there's enough of the product in stock, your purchase will go through and you'll be prompted a bill! If the item is out of stock, you will go back to the main shopping menu.

There's also manager functionality, as well. Enter "node bamazonManager" to see a list of prompts that allow you to restock items with a low inventory!

Find-A-Friend

Full-Stacked web app powered by NodeJs, Express, Path and hosted by Heroku

Click here to check it out!

Click "Take the Quiz" on the home page to start the survey. After you answer questions where you give your opinion ranging from Strongly Agree (value of 1) to Strongly Disagree (value of 5) and "Submit", you will be shown a user who answered the survey similarly to you! Eat-Da-Burger

Node-mySQL-Express-Handlebars powered, Heroku hosted

Click here to check it out!

Easy to use Heroku-hosted App that allows you to write in the names of burgers and make them with the "make Burger" button, then click on the button next to any burger in the left-hand column of the screen to "eat" that burger. All eaten burgers appear on the right-hand column. This app uses Node JS, Express, Express-Handlebars, Path JS, mySQL, and a self-made ORM to handle mySQL-to-JS data parsing
